CSS Style Basics Cheatsheet

CSS can be used to style a web page by appling color to the background and the text shown on the screen.

CSS can also be used to change fonts,sizes, backgrounds and much more.

The below table will show all the comands and theire propeties

Tag Name Description Exemple
color: color Used in CSS to aply colore to the text,from the h1...h6 and p tags as well color: red
background-color: background color Used in CSS to set the background color.Can be applayed behaind just a p or h1....h6 or over the hole body/header/footer background-color: gold
font-family: font Used in CSS to set the font of the text.Can be used on all tags from h1....h6 adn p font-family: Ariel or 'Times new Roman'--->for fonts of more words
font-size: font size Used in CSS to set the size of the font. font-size: small,midium,big
font-weight: thickness Used in CSS to give words the thickness desired. font-weight: bold,bolder,light
